## **üåê Overview**  
**STAGE-TELECOM** is a **MERN stack** application designed for **Tunisie Telecom** to manage:  
‚úî **Base stations (BTS)**  
‚úî **Antennas and transmissions**  
‚úî **Network issues and tickets**  
‚úî **Integrated Power BI dashboard**  

**Key Goals**:  
- Digitize telecom network management in Tunisia  
- Optimize real-time incident tracking  
- Provide automated reports for decision making

## **üöÄ Getting Started**  

### **üìã Prerequisites**  
- **Node.js** (v18+)  
- **MongoDB** (Atlas or local)  
- **Git**  

### **üíª Installation**  
1. **Clone the repository**:  
   ```bash
   git clone https://github.com/sarra-bennour/stage-telecom
   ```  
2. **Navigate to the project directory**:  
   ```bash
   cd stage-telecom
   ```  
3. **Install dependencies**:  
   ```bash
   npm install
   ```  

### **‚ö° Usage**  
- **Start the development server**:  
  ```bash
  npm start
  ```  
- **Access the app**:  
  `http://localhost:3000`  

### **üß™ Testing**  
Run the test suite with:  
```bash
npm test
```
